**Personal Portfolio Website**

This project is a modern, Django-based portfolio website designed to present my work, skills, and professional journey in a clean and structured way. It reflects my preference for simplicity, clarity, and a warm visual style, highlighted by a subtle caramel accent theme.

The goal of this portfolio is not just to showcase projects, but to create a space that communicates who I am, what I build, and how I approach problem-solving.

**Features**

The application includes several core pages that provide a complete overview of my profile and work:

A Home page that introduces me and sets the tone of the site
An About page that gives more insight into my background and interests
A Services section outlining what I can offer
A Projects page where I showcase selected work and practical implementations
A Contact page for easy communication

Behind the scenes, the platform is powered by dynamic Django models, allowing content to be easily updated and managed. These include:

Profile model for personal and professional details
Project model to organize and display my work
Skills model to highlight my technical capabilities

The interface is built with Bootstrap 5, ensuring the site is fully responsive and accessible across different devices.

**Purpose**

This portfolio serves as a central place to document my growth, share my work, and present myself professionally to potential collaborators, employers, and clients.
